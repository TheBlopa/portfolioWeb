App Recetas de Comida

Proyecto personal para recoger y usar mi lista de recetas, que se actualiza cuando quira cocinar recetas diferentes. Realizada con Python, kivy como interfaz, firebase y buildozer para conseguir la versión android. Este proyecto nació de las cenizas de un trabajo de la carrera para crear una aplicación para alertar si un bebe tendrá o tiene fiebre obteniendo datos de sensores de presión y temperatura.
